¿Qué son los Genéricos en Java?

Los genéricos en Java, introducidos en Java 5, permiten definir clases, interfaces y métodos con tipos de datos parametrizados. Esto proporciona mayor seguridad de tipos en tiempo de compilación y elimina la necesidad de realizar conversiones explícitas (casting). Los genéricos hacen que el código sea más reusable, legible y menos propenso a errores.

En este tutorial, exploraremos los siguientes conceptos:

  • Clases genéricas
  • Interfaces genéricas
  • Métodos genéricos
  • Wildcards (comodines)
  • Tipos acotados
  • Ejemplos prácticos

1. Clases Genéricas

Una clase genérica es una clase que puede trabajar con cualquier tipo de datos especificado al instanciarla. Se define utilizando un parámetro de tipo, generalmente denotado por <T>.


class Contenedor<T> {
    private T contenido;
    public Contenedor(T contenido) { this.contenido = contenido; }
    public T obtener() { return contenido; }
    public void establecer(T nuevoContenido) { this.contenido = nuevoContenido; }
}

En este ejemplo, Contenedor<T> puede almacenar cualquier tipo de dato, como String o Integer.

2. Interfaces Genéricas

Las interfaces genéricas permiten definir contratos con tipos parametrizados. Una clase que implemente esta interfaz debe especificar el tipo concreto.


interface Procesador<T> {
    T procesar(T entrada);
}

3. Clases con Múltiples Parámetros de Tipo

Útil para estructuras como pares clave-valor.


class Par<K, V> {
    private K clave;
    private V valor;
    public Par(K clave, V valor) { this.clave = clave; this.valor = valor; }
    public K getClave() { return clave; }
    public V getValor() { return valor; }
}

4. Métodos Genéricos

Permiten definir métodos parametrizados y usar wildcards.


class Utilidades {
    public static <T> void imprimirArray(T[] array) {
        for (T elemento : array) System.out.print(elemento + " ");
        System.out.println();
    }
    public static void imprimirLista(List<?> lista) {
        for (Object e : lista) System.out.print(e + " "); System.out.println();
    }
    public static double sumarNumeros(List<? extends Number> lista) {
        double suma = 0; for (Number n : lista) suma += n.doubleValue(); return suma;
    }
    public static void agregarNumeros(List<? super Integer> lista) {
        for (int i = 1; i <= 5; i++) lista.add(i);
    }
}
  • imprimirArray: Método genérico.
  • imprimirLista: Wildcard ?.
  • sumarNumeros: Wildcard ? extends Number.
  • agregarNumeros: Wildcard ? super Integer.

5. Tipos Acotados

Restringen tipos a subclases específicas.


class Calculadora<T extends Number> {
    private T numero;
    public Calculadora(T numero) { this.numero = numero; }
    public double getValorDoble() { return numero.doubleValue(); }
    public boolean esMayorQue(T otro) { return numero.doubleValue() > otro.doubleValue(); }
}

6. Ejemplo Completo de Uso


import java.util.*;
public class EjemploGenericos {
    public static void main(String[] args) {
        Contenedor cs = new Contenedor<>("Hola");
        System.out.println(cs.obtener());
        Contenedor ci = new Contenedor<>(42);
        System.out.println(ci.obtener());
        Par p = new Par<>("edad", 25);
        System.out.println(p.getClave()+":"+p.getValor());
        Integer[] arr = {1,2,3}; Utilidades.imprimirArray(arr);
        List l = Arrays.asList(1,2,3); Utilidades.imprimirLista(l);
        System.out.println(Utilidades.sumarNumeros(l));
        List ln = new ArrayList<>(); Utilidades.agregarNumeros(ln);
        System.out.println(ln);
        Calculadora calc = new Calculadora<>(10);
        System.out.println(calc.getValorDoble()+" "+calc.esMayorQue(5));
    }
}
